What is GITANALYTICS?

GitAnalytics is a website that provides some analyses of pull requests in GitHub


The project must be public and have at leat one pull request
You can show the analyse for a incresable list of repositories (Sidebar > Analysis)
You can add a repository to analyse (Sidebar > Add repository)
Our agent fetch data from GitHub API every hour (XXh40)
When you add a new repository to fetch, please wait for your request to appear on the board below
